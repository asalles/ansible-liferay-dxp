ansible-liferay-DXP
========================

Ansible JBoss EAP 7.1 Standalone Playbook
Ansible Playbook for: JBoss EAP 7.1 Standalone + Liferay 7 + Oracle 12c

Adittional DB Drivers included:
- PostgreSQL
- HSQL
- MariaDB
- MySQL

------------

- ansible +2.7
- set inventory.
- set vars.
